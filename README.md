## Photothub

#### Unsplash clone using vanilla Javascript and the Unsplash API

## Getting Started

First, run the development server:

```bash
- npm i
- npm run serve
```

if you want to use sass

```bash
- npm run sass
```

Open [http://localhost:9000](http://localhost:9000) with your browser to see the result.
